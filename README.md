## Descripción del problema

## Instrucciones para ejecutar

2. Subir el archivo `precipitacion_2026enero__Medellin_AeropuertoEnriqueOlayaHerrera.csv` cuando se solicite en la celda de carga.
3. Ejecutar las celdas en orden de arriba hacia abajo.
